# React - Image Upload and Manipulation
This is an interview test project

## Quick Start
1. Install dependencies server & client
```bash
$ npm install
$ cd client
$ npm install
```

2. serve frontend on `localhost:3000` & backend on `localhost:5000` with one command
```bash
$ npm run dev
```
